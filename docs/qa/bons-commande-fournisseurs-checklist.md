# QA - Bons de commande fournisseurs

## Navigation

- [ ] `/fournisseurs` reste accessible.
- [ ] L'onglet Fournisseurs affiche la liste fournisseurs existante.
- [ ] L'onglet Bons de commande affiche les commandes fournisseurs.

## Creation commande

- [ ] Le bouton Nouveau bon de commande cree un brouillon.
- [ ] Le fournisseur peut etre selectionne.
- [ ] Le Projet ID peut etre renseigne.
- [ ] Le Chantier ID peut etre renseigne.
- [ ] La reference fournisseur peut etre renseignee.
- [ ] La date de livraison prevue peut etre renseignee.
- [ ] L'adresse de livraison chantier peut etre renseignee.

## Lignes document

- [ ] Un lot peut etre ajoute.
- [ ] Une ligne produit peut etre ajoutee.
- [ ] Une ligne prestation peut etre ajoutee.
- [ ] Quantite, unite, prix achat HT et TVA sont modifiables.
- [ ] Les totaux HT / TVA / TTC se recalculent.

## Statuts et exports

- [ ] Les statuts Brouillon, Envoye, Confirme, Livre partiellement, Livre et Annule sont disponibles.
- [ ] Le PDF se genere via document-engine.
- [ ] La preview s'affiche.
- [ ] L'assistant d'envoi s'ouvre.
- [ ] La sauvegarde persiste la commande.

## Rentabilite projet

- [ ] Un bon de commande avec Projet ID remonte dans l'onglet Rentabilite du projet.
- [ ] Un bon de commande avec Chantier ID lie au projet remonte aussi.
- [ ] Les achats HT impactent la marge brute.

## Build

- [ ] `npm run build` passe.
