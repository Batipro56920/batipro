# QA - Catalogue produits

## Navigation

- [ ] `/catalogue-produits` est accessible depuis la sidebar.
- [ ] La page charge une liste de produits.
- [ ] La page reste protegee par les permissions fournisseurs / approvisionnement.

## Fiche produit

- [ ] Creation produit fonctionne.
- [ ] Modification produit fonctionne.
- [ ] Suppression produit demande confirmation.
- [ ] Les champs designation, references, marque, categorie, unite, TVA, fournisseur, prix achat, prix vente et marge cible sont modifiables.

- [ ] Ajouter un prix negocie fonctionne.
