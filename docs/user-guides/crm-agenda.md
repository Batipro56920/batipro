# Guide utilisateur — Agenda commercial CRM

L’Agenda commercial regroupe les rendez-vous, relances et tâches CRM.

## Header

- **Tâche** : ouvre la création de tâche commerciale.
- **RDV** : ouvre la création de rendez-vous.
- **Relance** : ouvre la création de tâche de relance.
- **Import calendrier** : action préparée, désactivée tant que l’import n’est pas finalisé.

## KPI

- **Tâches aujourd’hui** : tâches dues aujourd’hui.
- **RDV aujourd’hui** : rendez-vous du jour.
- **Relances dues** : relances non terminées.
- **En retard** : actions dépassées.
- **Cette semaine** : événements à venir sur 7 jours.

## Calendrier

Trois vues sont disponibles :

- Jour,
- Semaine,
- Mois.

Les couleurs indiquent le type :

- bleu : rendez-vous,
- orange : relance,
- vert : terminé,
- rouge : urgent ou en retard.

## Action center

Le panneau de droite regroupe :

- Aujourd’hui,
- En retard,
- Cette semaine,
- Relances à appeler.

## Drawer événement

Un clic sur un événement ouvre un drawer avec :

- Détail,
- Notes,
- Historique.

Actions disponibles :

- Terminer pour les tâches,
- Reporter et Annuler sont préparés mais désactivés.
