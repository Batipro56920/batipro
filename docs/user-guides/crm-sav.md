# Guide utilisateur — Service après-chantier CRM

Le module SAV permet de suivre les tickets après chantier, les interventions et les résolutions.

## Header

- **Nouveau ticket** : ouvre la création de ticket SAV.
- **Export** : action préparée, désactivée tant que l’export n’est pas finalisé.
- **Modèles réponses** : action préparée pour les réponses client.

## KPI

- **Tickets ouverts** : tickets non clos.
- **Urgents** : tickets à priorité élevée.
- **En attente client** : tickets bloqués côté client.
- **En intervention** : tickets planifiés ou en cours.
- **Fermés ce mois** : tickets clos sur le mois courant.
- **Délai moyen** : métrique à connecter.

## Toolbar

Recherche par ticket, client ou chantier.

Filtres :

- client,
- chantier,
- priorité,
- statut,
- intervenant,
- date.

## Vues

- **Liste** : vue tableau complète.
- **Kanban** : tickets par statut.
- **Planning interventions** : tickets triés par intervention planifiée.

## Drawer ticket

Un clic sur un ticket ouvre un drawer avec :

- Détail,
- Historique,
- Photos,
- Messages,
- Intervention,
- Documents,
- Clôture.

Actions préparées :

- Assigner,
- Planifier,
- Message client,
- Clôturer.
