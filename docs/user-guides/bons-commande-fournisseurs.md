# Guide utilisateur - Bons de commande fournisseurs

Les bons de commande fournisseurs sont accessibles depuis le module Fournisseurs, onglet Bons de commande.

## Objectif

Le bon de commande formalise un achat ou une prestation fournisseur et alimente la rentabilite projet.

Chaque bon de commande peut etre lie a :

- un fournisseur ;
- un projet ;
- un chantier ;
- un lot ou une prestation ;
- une adresse de livraison chantier.

## Statuts

Les statuts disponibles sont :

- Brouillon ;
- Envoye ;
- Confirme ;
- Livre partiellement ;
- Livre ;
- Annule.

## Contenu du document

L'editeur utilise le document-engine commun a Batipro.

Il permet de renseigner :

- date de commande ;
- date de livraison prevue ;
- reference fournisseur ;
- lignes produit ou prestation ;
- prix achat HT ;
- TVA ;
- total TTC.

## Rentabilite projet

Quand le bon de commande contient un `Projet ID` ou un `Chantier ID` lie au projet, son montant HT remonte dans l'onglet Rentabilite du projet.

Cette V1 prepare la future liaison avec :

- catalogue produits ;
- prix fournisseurs negocies ;
- bons de livraison ;
- factures fournisseurs.
