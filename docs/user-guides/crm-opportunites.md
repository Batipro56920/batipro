# Guide utilisateur — Opportunités CRM Batipro

La page **Opportunités** permet de piloter le pipeline commercial depuis le premier contact jusqu’à la signature.

## Header

- **Nouvelle opportunité** : ouvre le formulaire existant de création d’opportunité.
- **Import** : action préparée, désactivée tant que l’import n’est pas finalisé.
- **Filtres** : les filtres sont disponibles directement dans la toolbar.

## KPI

- **Opportunités ouvertes** : affaires en cours.
- **CA pipeline** : total estimé des affaires ouvertes.
- **CA pondéré** : montant estimé pondéré par probabilité.
- **Taux conversion** : ratio affaires gagnées / total.
- **Signature ce mois** : opportunités gagnées sur le mois courant.

## Toolbar

La toolbar permet de filtrer :

- par recherche texte,
- par commercial,
- par source,
- par budget,
- par date de création,
- par température commerciale : chaud, tiède, froid, gagné, perdu.

## Pipeline

Le pipeline affiche les étapes commerciales :

- Lead,
- Qualification,
- Visite,
- Chiffrage,
- Devis envoyé,
- Négociation,
- Signature,
- Gagné,
- Perdu.

Chaque colonne indique le nombre d’affaires, le montant total et contient ses opportunités.

## Drag & drop

Une carte peut être déplacée vers une autre colonne. Le déplacement utilise la logique existante de mise à jour d’étape.

## Cartes opportunités

Chaque carte affiche :

- nom de l’affaire,
- client ou prospect,
- montant,
- probabilité,
- échéance,
- commercial,
- badge priorité,
- prochaine action.

## Drawer détail

Un clic sur une carte ouvre un drawer latéral avec :

- informations client/prospect,
- montant,
- probabilité,
- échéance,
- notes,
- activité et historique de base.

Les tâches, relances, devis liés et historique détaillé sont prévus pour une étape suivante.
