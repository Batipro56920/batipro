# Guide utilisateur — Clients CRM Batipro

La page **Clients** centralise les informations commerciales et opérationnelles liées aux clients.

## Header

- **Ajouter client** : ouvre le formulaire de création client existant.
- **Import** : action préparée, désactivée tant que l’import n’est pas finalisé.
- **Fusion doublons** : action préparée, désactivée tant que la fusion n’est pas sécurisée.

## KPI

- **Clients actifs** : clients non archivés.
- **Nouveaux ce mois** : clients créés sur les 30 derniers jours.
- **CA total** : montant des chantiers liés.
- **Chantiers actifs** : chantiers en préparation, en cours ou en pause.
- **SAV ouverts** : tickets SAV non clos.
- **Factures en attente** : factures non payées.

## Toolbar

La recherche couvre le nom, email, téléphone, société, ville et notes.

Filtres disponibles :

- type client,
- commercial,
- statut,
- chantier actif,
- SAV,
- date de création.

## Vues

- **Liste** : vue complète et dense.
- **Cartes** : lecture rapide par client.
- **Activité** : clients triés par mise à jour récente.

## Liste clients

La liste affiche :

- client,
- type,
- contact,
- commercial,
- nombre de devis,
- chantiers,
- CA,
- statut,
- actions.

## Drawer client

Un clic sur un client ouvre une fiche rapide avec :

- identité,
- commercial,
- devis,
- chantiers,
- facturation,
- SAV,
- documents,
- historique,
- notes.

## Bonnes pratiques

- Garder les coordonnées client complètes.
- Relier les devis et chantiers au bon client.
- Surveiller les SAV ouverts et factures en attente.
- Archiver uniquement les clients réellement inactifs.
