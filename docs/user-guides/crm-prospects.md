# Guide utilisateur — Prospects CRM Batipro

La page **Prospects** sert à gérer les leads, demandes entrantes et contacts commerciaux avant conversion en client, opportunité ou devis.

## KPI

- **Prospects actifs** : prospects encore ouverts.
- **Nouveaux semaine** : prospects créés sur les 7 derniers jours.
- **À relancer** : prospects non gagnés, non perdus et non archivés.
- **Sans activité > 7j** : prospects ouverts sans mise à jour récente.
- **Budget moyen** : moyenne des budgets renseignés.
- **Conversion clients** : part des prospects convertis en clients.

## Filtres

La barre de filtres permet de rechercher instantanément par nom, email, ville, projet, statut ou source.

Les filtres rapides permettent de passer rapidement entre :

- tous les prospects,
- à relancer,
- chauds,
- sans suite,
- convertis.

Les filtres avancés couvrent le statut, la source, le commercial, le budget, la date de création et la relance due.

## Vues

Trois vues sont disponibles :

- **Liste** : tableau complet pour gérer beaucoup de prospects.
- **Kanban** : regroupement par statut commercial.
- **Cartes** : vue visuelle compacte pour lecture rapide.

## Ajouter un prospect

Le bouton **Ajouter prospect** ouvre le formulaire existant de création prospect.

## Actions commerciales

Depuis une ligne ou une carte prospect :

- **Appeler** : lance un appel si un numéro existe.
- **Email** : ouvre un email si une adresse existe.
- **Tâche** : crée une tâche de relance.
- **Opportunité** : ouvre la création d’opportunité.
- **Devis** : crée un devis brouillon.
- **Convertir** : transforme le prospect en client.

Les actions non finalisées sont désactivées avec une indication claire.

## Drawer fiche rapide

Un clic sur un prospect ouvre une fiche rapide latérale avec :

- coordonnées,
- projet,
- budget,
- source,
- statut,
- notes,
- historique basique,
- actions principales.

## Bonnes pratiques commerciales

- Qualifier rapidement les nouveaux prospects.
- Ajouter une tâche de relance dès qu’un contact n’est pas traité.
- Convertir en client uniquement quand le contact devient exploitable commercialement.
- Créer une opportunité pour suivre une affaire avec montant et probabilité.
- Créer un devis lorsque le besoin est suffisamment défini.
